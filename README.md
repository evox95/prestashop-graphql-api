PrestaShop Front GraphQL API
---

Lightweight implementation of the GraphQL API for PrestaShop front-office.

## Requirements

* PHP >= 7.4
* PrestaShop >= 1.7.8.0

## Usage

Put files in the `/modules/api_graphql` directory.
